# 💰 Кошелёк — Личные Финансы

Минималистичное PWA-приложение для учёта личных финансов.  
**100% офлайн**, без серверов, без аккаунтов, без VPN.

---

## 🚀 Деплой на GitHub Pages (пошагово)

### Шаг 1: Создай репозиторий на GitHub

1. Зайди на [github.com/new](https://github.com/new)
2. **Repository name**: `koshelek` (или любое другое)
3. **Public** (чтобы Pages работал бесплатно)
4. **Не ставь** галочки на README, .gitignore, license (они уже есть)
5. Нажми **Create repository**

### Шаг 2: Залей код

Открой терминал в папке проекта и выполни:

```bash
git init
git add .
git commit -m "🚀 Кошелёк v1.0.0"
git branch -M main
git remote add origin https://github.com/ТВОЙ_ЮЗЕРНЕЙМ/koshelek.git
git push -u origin main
```

> ⚠️ Замени `ТВОЙ_ЮЗЕРНЕЙМ` на свой GitHub username

### Шаг 3: Включи GitHub Pages

1. Зайди в репозиторий на GitHub
2. **Settings** (вкладка сверху)
3. **Pages** (в левом меню)
4. **Source**: выбери **GitHub Actions**
5. Всё! Деплой запустится автоматически

### Шаг 4: Дождись деплоя

1. Зайди во вкладку **Actions** в репозитории
2. Дождись пока workflow станет зелёным ✅ (1-2 минуты)
3. Сайт готов по адресу:

```
https://ТВОЙ_ЮЗЕРНЕЙМ.github.io/koshelek/
```

### Шаг 5: Установи на iPhone

1. Открой ссылку в **Safari** на iPhone
2. Нажми **Поделиться** (квадрат со стрелкой вверх)
3. **На экран «Домой»**
4. Готово! Работает как обычное приложение, без VPN, офлайн

## 🔄 Как обновить

1. Внеси изменения в код
2. ```bash
   git add .
   git commit -m "Описание изменений"
   git push
   ```
3. GitHub автоматически пересоберёт и задеплоит

---

## 💾 Перенос данных при обновлении

1. **До обновления**: Настройки → Экспорт данных → сохранить файл
2. Обнови приложение (или переустанови)
3. **После**: Настройки → Импорт данных → выбрать файл
4. Все данные на месте ✅
